# 目录重组说明

## 方法一：使用Python脚本（推荐）

我已经创建了 `重组目录.py` 脚本，你可以直接运行它：

1. **打开命令提示符或PowerShell**
2. **切换到项目目录**：
   ```bash
   cd "d:\学习\中山大学智能工程学院本科生课程作业"
   ```
3. **运行脚本**：
   ```bash
   python 重组目录.py
   ```

脚本会自动：
- 创建所有课程目录（如果不存在）
- 将文件从学期目录移动到对应的课程目录
- 删除空的学期目录

1. 打开 `大三上\分布式计算` 目录
2. 全选所有文件和文件夹（Ctrl+A）
3. 剪切（Ctrl+X）
4. 打开项目根目录下的 `分布式计算` 目录
5. 粘贴（Ctrl+V）

重复以上步骤，将所有课程的文件都移动到对应的课程目录。

## 注意事项

1. **备份**：操作前建议先备份整个目录
2. **重复文件**：如果目标目录已存在同名文件，脚本会跳过并提示
3. **压缩包**：`.rar` 和 `.zip` 文件也会被移动，后续可以删除
4. **临时文件**：建议移动完成后清理临时文件（如 `新建文件夹`、`新建 Microsoft Word 文档.docx` 等）

## 问题排查

如果遇到问题：
1. 检查Python是否已安装：`python --version`
2. 检查路径是否正确
3. 检查文件权限
4. 查看脚本输出的错误信息
