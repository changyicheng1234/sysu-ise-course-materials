# 中山大学智能工程学院课程资料共享

> 📚 这是一个课程资料共享项目，旨在收集和整理各门课程的学习资料，方便学弟学妹查找和使用。

## ✨ 特性

- 📖 **丰富的课程资料**：涵盖多个专业的核心课程
- 🤖 **自动更新**：GitHub Actions 自动维护 README，无需手动编辑
- 🔍 **智能分类**：根据文件命名自动分类（作业、实验、报告等）
- 📝 **选课建议**：每门课程都有详细的选课建议和学习经验

## 🚀 如何使用

### 查找资料

1. 进入对应的课程目录
2. 查看 `README.md` 了解课程信息和资料列表
3. 根据需要下载或查看资料

### 添加资料

1. **Fork 本项目**
2. **添加资料**：将文件放入对应课程目录
3. **运行更新脚本**：运行 `更新README资料.py` 更新资料列表
4. **编辑选课建议**（可选）：手动编辑 README 的"选课建议"部分
5. **提交更改**：提交到 GitHub

详细说明请查看 [贡献指南](CONTRIBUTING.md) 和 [使用说明](使用说明.md)

## 🤖 自动化说明

本项目使用本地脚本自动更新 README：

- ✅ **自动更新资料列表**：运行脚本自动扫描并更新"已有资料"部分
- ✅ **智能分类**：根据文件名自动识别文件类型
- ✅ **保留选课建议**：脚本只更新资料列表，"选课建议"部分需要手动编辑
- 📝 **使用方式**：添加资料后，运行 `更新README资料.py` 或双击 `运行更新README.bat`

## 📋 文件命名建议

为了让脚本更好地识别文件，建议使用以下命名：

- **作业**：`作业1.pdf`、`homework2.docx`
- **实验**：`实验1/`、`exp2/`、`lab3/`
- **期末**：`期末报告.pdf`、`final_exam/`
- **报告**：`课程报告.docx`、`report.pdf`
- **课件**：`第1章.ppt`、`lecture_slides.pptx`
