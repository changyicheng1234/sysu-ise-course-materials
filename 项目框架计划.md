# 中山大学智能工程学院课程攻略共享计划 - 框架计划

## 📋 项目概述

整理和共享中山大学智能工程学院本科生课程的学习资料，包括作业、实验报告、复习资料等。

## 📁 目录结构

### 顶层结构

```
中山大学智能工程学院课程攻略/
├── README.md                    # 项目主README
├── LICENSE                      # 开源许可证
├── .gitignore                   # Git忽略文件配置
│
├── 分布式计算/                   # 按课程名称组织
│   ├── README.md                # 课程README（目录使用方式 + 选课建议）
│   ├── 作业/
│   ├── 实验/
│   ├── 复习资料/
│   ├── 试卷/
│   └── 其他/
│
├── 自动控制原理/
│   ├── README.md
│   ├── 作业/
│   └── ...
│
└── [其他课程]/
```

### 课程目录命名

- **使用课程全称**：如 `分布式计算`、`自动控制原理`
- **避免缩写**：除非是官方通用缩写
- **统一格式**：使用中文课程名称

### 子目录结构（可选）

每个课程目录下可以包含以下子目录（根据实际资料情况）：

- `作业/` - 平时作业
- `实验/` - 实验报告和代码
- `复习资料/` - 复习笔记、总结
- `试卷/` - 历年试卷
- `教材/` - 电子版教材（可选）
- `其他/` - 其他相关资料

## 📝 课程README模板

每个课程目录下需要包含一个 `README.md` 文件，内容只需包含两部分：

### 1. 目录使用方式

说明这个课程目录下有哪些资料，如何查找和使用。

### 2. 选课建议

对这门课程的教师评价和选课建议。

**简化模板**：

```markdown
# [课程名称]

## 目录说明

本目录包含以下资料：

- **作业**：位于 `作业/` 目录，包含[说明]
- **实验**：位于 `实验/` 目录，包含[说明]
- **复习资料**：位于 `复习资料/` 目录，包含[说明]
- **试卷**：位于 `试卷/` 目录，包含[说明]

## 选课建议

### [教师姓名拼音首字母缩写，如：ZMQ]

- **推荐指数**：⭐⭐⭐⭐⭐ (1-5星)
- **授课风格**：[描述]
- **作业要求**：[描述]
- **考试难度**：[描述]
- **给分情况**：[描述]
- **选课建议**：[建议选课/谨慎选课/不推荐]

### [其他教师]

[同上格式]
```

## 📋 文件命名规范

- **作业文件**：`作业X-学号-姓名.pdf` 或 `作业X-学号-姓名.docx`
- **实验报告**：`实验X-学号-姓名.pdf` 或 `实验X-学号-姓名.docx`
- **试卷文件**：`试卷-学年-学期.pdf`
- **避免使用**：`新建文件夹`、`新建 Microsoft Word 文档` 等默认名称

## 🔄 迁移计划

### 第一步：按课程重新组织

将现有按年级组织的资料重新分类到课程目录：

**当前结构**：
```
大三上/
  ├── 分布式计算/
  ├── 图像处理/
  └── ...
```

**目标结构**：
```
课程目录/
  ├── 分布式计算/
  ├── 图像处理/
  └── ...
```

### 第二步：清理文件

- 删除 `.rar`、`.zip` 压缩包（解压后保留内容）
- 删除临时文件（如 `新建文件夹`、`新建 Microsoft Word 文档.docx`）
- 删除重复文件
- 删除或替换个人隐私信息（学号、姓名等，可使用占位符）

### 第三步：统一文件命名

按照文件命名规范重命名文件。

### 第四步：创建课程README

为每个课程目录创建 `README.md`（可以先创建空文件或简单说明，后续统一填写）。
